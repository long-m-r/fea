Plotting Functions¶

fea.plot.plotting.plot(graph, **kwargs)¶

Plots a 2d or 3d graph given an FEA LatticeGraph object (the output of the fea function)

Parameters

graph – FEA Lattice Graph to Plot

Named Arguments:

figure: The figure to plot on cmap: color map (defaults to inferno) x: specify variable name for X-axis y: specify variable name for Y-axis z: specify variable name for Z-axis c: specify variable name for color-axis vertices: Default True. Whether or not to plot vertices edges: Default True. Whether or not to plot edges

Returns

Tuple of matplotlib figure and plot

Example

solution = fea(model, [x,y,z]) solfigure, solplot = plot(solution, x=’x’, y=’y’, z=’z’) solfigure.savefix(‘solution_figure.svg’)
